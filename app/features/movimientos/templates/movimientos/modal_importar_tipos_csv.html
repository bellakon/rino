Importar Tipos de Movimientos desde CSV
Formato del archivo CSV:
  • Columnas: id, nomenclatura, nombre, descripcion, categoria, campos_personalizados, activo
  • Si el ID existe, se actualiza; si no, se crea nuevo
  • campos_personalizados debe ser JSON válido o vacío
  • Categorías: Comisión, Licencia, Permiso, Autorización, Capacitación, Otros
